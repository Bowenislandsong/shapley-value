# Shapley Value Calculator - Project Webpage

This directory contains the project webpage for the Shapley Value Calculator package.

## About

The webpage is hosted via GitHub Pages and provides:
- Project overview and features
- Installation instructions
- Quick start guide
- Performance benchmarks
- Use cases
- Links to documentation and examples
- Author information

## Viewing the Webpage

Once GitHub Pages is enabled in the repository settings:
- The webpage will be available at: `https://bowenislandsong.github.io/shapley-value/`
- GitHub Pages should be configured to deploy from the `docs/` directory

## Local Development

To preview the webpage locally:

```bash
cd docs
python3 -m http.server 8080
```

Then open http://localhost:8080/ in your browser.

## Deployment

The webpage is automatically deployed to GitHub Pages via the `.github/workflows/pages.yml` workflow when changes are pushed to the `main` branch.
